2048

by: Alvontay Maher

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

It just seemed like a simple project where the difficulty would be from making sure it works as its supposed to, as opposed to making sure it would work at all.

2. What I Learned

One thing I had learned from user testing was that on my own there were some functions of the game that I had implemented incorrectly such as the order tiles combine in, or when not to add new tiles to the game board.

3. Architecture of My Game

The majority of the game is a 4x4 grid of tiles. The grid starts empty but as you move tiles around it gets filled more and more. if the values of two tiles are the same then they can combine into a single tile with double their values. The goal of the game is to combine tiles until you have a tile with the value of 2048. You lose the game if the board fills up and you cannot combine anything to clear up space. The main two classes I created were the class containing the game board and the functions for moving and combining, and the Tile class which drew each tile and held their values.

4. Technical Implementation

One of the parts I'm proud of is the method where I check whether the player had lost or not. I'm proud of it because Initially I had though I would need to basically make a copy of the games state and then check if the player could make any moves in that state, however I was able to simplify it by going through each tile checking the values of the adjacent tiles to see if they could combine.

5. Most Challenging Part

The most challenging part of this project was working to combine the functions I had created for moving and combining tiles. The main difficulty in this was that I needed to use different conditions depending on the direction the player had pressed. I had fixed this by using negative and positive variables for some and ternary (statements?) for others.

6. Future Fixes and Updates

I would just fix one of the logic errors that I had broken at some point.
